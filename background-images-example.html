Example 1: Simple Background Image

Replace 'your-image.jpg' with your image path

background-image: url('your-image.jpg');

Example 2: Animated Background

Background image with subtle movement animation

animation: bgMove 20s ease-in-out infinite;

Example 3: Gradient Overlay

Background image with gradient overlay using ::before

opacity: 0.3; /* Adjust for transparency */

Example 4: Multiple Backgrounds

Gradient + background image combined

background: linear-gradient(...), url('your-image.jpg');

Example 5: Parallax Effect

Fixed background for parallax scrolling effect

background-attachment: fixed;

Example 6: Video Background

Video background with overlay content

<video autoplay muted loop>

Example 7: SVG Pattern

Custom SVG pattern as background

background-image: url('data:image/svg+xml,...');

Example 8: Animated SVG

SVG with built-in animations

<animate attributeName="r" values="2;4;2" dur="3s"/>
